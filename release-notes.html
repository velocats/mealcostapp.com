Release Notes

What’s new in MealCost. Short, scannable updates you can check before (or after) an upgrade.

v1.0.6

v1.0.6

February 2026

Cleaner flows, clearer charts, and better sample data so the dashboards and insights look great right away.

Dashboard

  • New shareable stat: “Dining out costs X× more on average” callout.
  • Improved chart math consistency so “Home vs Dining Out” matches dashboard totals.
  • Timeframes now behave consistently across free vs paid (free uses a rolling 30 days; paid can use multiple ranges).

Insights

  • Filter controls tightened up to avoid big, interrupting selection screens.
  • Home vs Eating Out calculations corrected so dining-out meals reliably appear.
  • Timeframe labels standardized back to the simple Day / Week / Month / Year style.

Meals

  • Streamlined “Make a Meal” flow with less UI jumping and fewer taps to get to “Add Items”.
  • Tighter ingredient list layout so more items fit on screen at once.

Receipts

  • Streamlined receipt add/edit flow to reduce friction when adding grocery items.
  • Sample receipts and items updated so lists and charts populate immediately.

Sample data + chart quality

  • Sample history expanded to span multiple years so Year charts look meaningful.
  • More variation between home-cooked vs dining out sample data to make trends visible.
  • General fixes for empty charts, missing series, and outlier spikes.
